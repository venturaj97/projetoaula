# Validador de CPF, Nome e Estado

Este é um mini projeto em Python utilizando Tkinter para criar uma interface gráfica que permite validar CPF, nome e estado.

## Funcionalidades

- **Validação de CPF:** Verifica se o CPF inserido é válido.
- **Validação de Nome:** Verifica se o nome inserido é válido.
- **Validação de Estado:** Verifica se o estado inserido é válido.

## Requisitos

- Python 3.x instalado.
- Bibliotecas necessárias:
  - tkinter
  - sqlite3
  - PIL (Python Imaging Library)

## Como Executar

1. Certifique-se de ter os requisitos listados acima instalados em seu ambiente Python.
2. Clone ou baixe este repositório.
3. Execute o script `main.py`.

```python
python main.py
